# Guide Copywriting

## Principe central

La copie doit parler du moment vecu par l'enfant, pas de la technologie. L'IA est un moyen, pas la promesse.

## Hero recommande

Titre :

> Une histoire douce pour aider votre enfant a vivre les petits grands moments.

Sous-titre :

> Choisissez une situation, ajoutez quelques details sur votre enfant, et recevez une histoire personnalisee ou il devient le heros.

CTA principal :

> Creer une histoire

CTA secondaire :

> Voir un exemple

## Messages cles

- Chaque enfant vit des petits grands moments : une rentree, une peur, une colere, un changement.
- StoryKid transforme ces moments en histoires personnalisees, douces et rassurantes.
- Pense pour les parents qui veulent accompagner avec douceur, sans ecran complique ni application a installer.
- Quelques details suffisent pour creer une histoire qui ressemble a votre enfant.

## Microcopy formulaire

- "Quelques details suffisent. Vous pourrez verifier avant de generer."
- "Choisissez le ton qui ressemble le mieux a votre rituel du soir."
- "Quel message aimeriez-vous que l'histoire transmette doucement ?"
- "Votre histoire est prete a prendre forme."

## CTA

Bons CTA :

- Creer une histoire.
- Generer mon histoire.
- Lire l'histoire demo.
- Creer une autre histoire.
- Voir les offres.

CTA a eviter :

- Soumettre.
- Envoyer.
- Optimiser.
- Demarrer l'IA.
