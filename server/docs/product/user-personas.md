# Personas

## Parent presse

Parent de 30 a 42 ans, actif, peu disponible le soir. Il veut faire bien, mais n'a pas toujours l'energie de chercher un livre adapte ou d'inventer une histoire.

Motivations :

- Trouver rapidement les bons mots.
- Creer un moment calme avant le coucher.
- Personnaliser sans passer une heure sur un outil.

Objections :

- Peur que l'IA produise un texte froid.
- Doute sur la qualite pedagogique.
- Reticence a donner trop d'informations personnelles.

## Parent attentif aux emotions

Parent sensible a l'education positive, aux livres jeunesse et aux rituels familiaux. Il cherche un support doux pour parler d'une emotion.

Motivations :

- Aider l'enfant a se reconnaitre dans une histoire.
- Ouvrir une discussion.
- Garder un souvenir personnalise.

Objections :

- Refus d'un ton trop medical.
- Besoin d'un produit beau et credible.
- Attente forte sur la justesse du texte.

## Parent cadeau

Grand-parent, parrain, marraine ou proche qui veut offrir un livre personnalise.

Motivations :

- Faire un cadeau personnel.
- Creer un souvenir durable.
- Commander un livre imprime plus tard.

Objections :

- Besoin d'un parcours tres simple.
- Doute sur les delais et la qualite d'impression.
