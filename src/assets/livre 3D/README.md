# Book Generic Assets - StoryKid

Place ce dossier dans `/public/assets/book-generic/`.

Ces assets sont neutres et réutilisables pour un viewer de livre générique. Codex doit injecter dynamiquement `book.coverImage` au-dessus du shell, puis afficher les pages/textes dans le template ouvert.

Ne pas générer un livre 3D par histoire : utiliser ce shell générique + couverture dynamique.
